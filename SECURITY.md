# Security Policy

## Supported Versions

We only provide security updates for the latest stable version of Party Land.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please **do not create a public issue**. Instead, please use the **Private Vulnerability Reporting** feature in the GitHub Security tab.

Alternatively, you can reach out to the lead developer on Farcaster: [@camilin23](https://warpcast.com/camilin23)

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary.

## Security of Private Keys

**WARNING:** Never share your private keys or seed phrases with anyone, including the project developers. Our team will NEVER ask for your PRIVATE_KEY or SEED_PHRASE.
